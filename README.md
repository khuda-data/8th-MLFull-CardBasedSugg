# 카드 데이터 기반 소비 취약군 식별 및 정책 제안  
### Suwon City Card Consumption Analysis

KHUDA 8기 ML Full 트랙 '연금술'팀 심화 프로젝트 결과물입니다.

### 프로젝트에 대한 간략한 설명

본 프로젝트는 **수원시 카드 소비 데이터를 활용하여 기존 복지·금융 지표로는 포착하기 어려운 소비 취약 집단을 식별하고, 데이터 기반 맞춤형 정책 방향을 제안**하는 것을 목표로 합니다.

행정동 × 연령대 단위의 소비 데이터를 기반으로 소비 구조, 시간대 특성, 소비 안정성 및 다양성 지표를 설계하였으며,  
**Isolation Forest와 군집 분석을 결합한 2단계 분석 프레임워크**를 적용해 서로 다른 원인을 가진 소비 취약군을 도출하였습니다.

분석 결과, 소비 취약성은 단순한 과소비 문제가 아니라 **생계 압박, 상권 환경(Place Effect), 노동·시간 구조** 등 상이한 요인에서 비롯됨을 확인하였으며, 이를 통해 **획일적 지원이 아닌 군집별 원인 기반 정밀 정책 설계의 가능성**을 제시합니다.
